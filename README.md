☢️ **XiEye(дословно: глаз Си Цзиньпиня) - Шутливая система по оценке социального рейтинга в телеграм канале.**

Система предоставлять отличный набор функционал для карательная действий во имя товарищ Xi и кошкодевка. А также поощрений за донос.
Функционал:
- Системное управление рейтинг (штраф и исключение из конференции за проступки при пониженном рейтинге)
- Пользовательское управление (Меню доносов, голосование, организация эвентов)
